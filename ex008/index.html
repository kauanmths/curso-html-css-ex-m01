Principais formataçãoes de texto

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag b. (não semântico).

Nesta frase temos um termo em destaque usando a tag strong.

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag i (não semântico).

Nesta frase temos um termo em itálico usando a tag em.

Texto marcado

Podemos também criar um texto marcado usando a tag mark.

Texto grande e pequeno

Estamos criando um texto grande com a tag big (não semântico) e um texto pequeno com a tag small.

Texto deletado

Podemos marcar um texto como excluído para indicar que deve ser lido, mas não considerado, com a tag del.

Texto inserido

Podemos marcar um texto como inserido para indicar que deve ser lido com atenção, com a tag ins.
